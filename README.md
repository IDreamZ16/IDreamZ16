# 💫 About Me:
My path to software development hasn't been a straight line, but it’s been a deliberate one. After building a strong foundation in Computer Science fundamentals, I took some time away to gain experience in various industries. That time away only solidified my passion for building tools that solve real problems.<br><br>Now, I’m back in the classroom refining my craft in Software Development and applying years of side-project experience to modern full-stack applications.
